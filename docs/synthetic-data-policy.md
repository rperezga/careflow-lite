# Synthetic Data Policy

careflow-lite is a **demo**. It uses **synthetic data only**.

- **No real patient data (PHI)** is ever stored or processed.
- This project is **NOT HIPAA-compliant** and must not be used for real healthcare data.
- All patient records are generated by the seed script and clearly labeled as demo data.
- The UI shows a persistent banner: "Synthetic data only. Not HIPAA-compliant."
